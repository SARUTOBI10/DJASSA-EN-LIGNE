# 🔐 ACCÈS MARCHAND CACHÉ

## Comment accéder au bouton marchand (réservé à l'administrateur)

Le bouton marchand est maintenant caché aux yeux des utilisateurs normaux pour des raisons de sécurité.

### Méthode d'accès :
1. **Raccourci clavier** : Appuyez sur `Alt + M` depuis n'importe quelle page
2. Le bouton marchand apparaîtra temporairement pendant 5 secondes
3. Cliquez dessus pour accéder à la page de connexion marchand

### Fonctionnalités :
- ✅ Bouton invisible pour les utilisateurs normaux
- ✅ Accessible uniquement via raccourci clavier connu
- ✅ Disparaît automatiquement après 5 secondes
- ✅ Reste visible quand le marchand est connecté (pour la déconnexion)

### Indicateurs subtils :
- Petit point orange très discret en haut à droite du bouton (opacity 0.3)
- Message dans la console développeur : "Appuyez sur Alt+M pour révéler le bouton marchand caché"

### Sécurité :
- Les utilisateurs normaux ne voient pas le bouton
- Aucun indice visuel clair sur son emplacement
- Seule la connaissance du raccourci permet l'accès

---
**Note :** Ce système garantit que seuls les administrateurs connaissant le raccourci peuvent accéder à l'interface marchand.
