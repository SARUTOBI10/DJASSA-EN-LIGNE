🆘 DÉPANNAGE
═════════════════════════════════════════════════════════════════════════════

❌ La page ne charge pas?
   → Assurez-vous que les 3 fichiers (HTML, CSS, JS) sont dans le même dossier
   → Vérifiez que les liens des fichiers sont corrects

❌ Mes données disparaissent au rechargement?
   → Vérifiez que localStorage n'est pas vidé par un autre paramètre du navigateur
   → Essayez d'utiliser une navigation privée

❌ Les emails ne s'envoient pas?
   → Actuellement, c'est normal, c'est une simulation
   → Les emails s'affichent dans la console (F12)
   → Pour les vrais emails, suivez EMAIL_INTEGRATION.md

❌ Les styles ne s'affichent pas?
   → Attendez le chargement complet de la page
   → Videz le cache du navigateur (Ctrl+Maj+Supp)
   → Vérifiez que style.css est bien chargé
